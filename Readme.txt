YetiCrab
Тестовое задание
Прототип системы ведения заявок для логистов в авто грузоперевозках в виде web-приложения. 
Прототип включает в себя вывод таблицы с заявками, счетчик заявок (количество заявок в таблице). 
Есть возможность перейти в режим администратора (без авторизации), где появляется возможность просмотра, редактирования и удаления заявок, создание новых заявок. 
В заявке есть следующие данные: 
● Номер заявки 
● Дата и время получения заявки от клиента 
● Название фирмы клиента 
● ФИО перевозчика 
● Контактный телефон перевозчика 
● Комментарии 
● ATI код сети перевозчика (кликабельно, переход на сайт). 
Ссылка такого вида: https://ati.su/firms/{ati}/info (пример: https://ati.su/firms/12345/info). 
Также реализована сортировка, поиск. Backend на Node Js и Express, frontend на React, база данных MongoDB.
